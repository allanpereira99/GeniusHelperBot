# geniushelperbot
## _geniushelperbot a bot based on chatgpt to facilitate some tasks._

## Installation

```sh
git clone https://github.com/allanpereira99/GeniusHelperBot.git
cd ./GeniusHelperBot
npm i
```
set environment variables in .env follow the example of .env.example.

## Build
```sh
npm run build
```
## Start
```sh
npm run start
```
